# Mobile App Usage Analysis

This project presents an interactive Power BI dashboard that analyzes mobile application usage data to understand user behavior and usage patterns.

## Project Overview
The dashboard provides insights into:
- Most frequently used mobile applications
- Usage trends across different time periods
- User engagement behavior

## Tools Used
- Power BI Desktop
- CSV dataset files

## Repository Structure
- Power BI report file (.pbix)
- Dataset files stored in the data folder
- Project documentation (README)

## How to Use
Download the `.pbix` file and open it using Power BI Desktop.  
Ensure the dataset files remain inside the `data` folder to allow proper data refresh.
